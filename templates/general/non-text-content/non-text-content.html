Non-text Content

All meaningful non-text content (images, video, audio, SVGs, icons and other content that isn't conveyed by text) needs to have alt text that provides the same information found in the content.

Purely decorative content (for example, a dividing line between a navigation bar and content) does not need alt text.

  1. Identify the non-text content (images, video, audio, SVGs, icons and other content that isn't conveyed by text)
  2. Decide if the content is meaningful or decorative.
  3. If meaningful see the following sections for instructions on how to provide alternative text.
  4. If decorative see the following sections for instructions on how to ensure they are ignored by assistive technologies.
