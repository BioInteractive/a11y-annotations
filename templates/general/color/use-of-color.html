Use of Color

Color must not be used as the only visual means of conveying information, indicating an action, prompting a response, or distinguishing a visual element.

  1. Identify Color-Reliant Information: Examine the design and identify any instances where color is used to convey information or distinguish visual elements. This may include things like:
    • Using red text to indicate errors.
    • Using green text to indicate success.
    • Using color-coded icons or labels to represent different categories or statuses.
    • Using color alone to indicate a link in a block of text.
  2. Check for Redundant Cues: Ensure that information or distinctions made through color are also conveyed using additional visual cues, such as text labels, icons, patterns, or shapes. This provides redundancy and ensures that people who cannot perceive color can still understand the content.

  • Color is used as the only visual means of conveying _________________.
  • Warning: Color must not be the only visual indication.
  • Recommendation: Add a non-color indicator such as an underline, symbol and/or text.
