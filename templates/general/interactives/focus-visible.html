Focus Visible

2.4.7 Focus Visible ensures that user interface components operable via keyboard have a visible focus indicator, so users can easily see where keyboard interactions will occur.

Benefits of Focus Visible

This success criterion is crucial for keyboard-only users, including people with motor disabilities, as it allows them to visually identify which element has focus. Additionally, users with cognitive disabilities, such as attention or memory limitations, benefit by easily discovering where focus is located, reducing confusion.

Examples of Focus Visibility

  • Text Fields: When a text field receives focus, a blinking caret appears, indicating the user can type, or the text is highlighted, showing it can be edited.
  • Interactive Elements: When a button or link receives focus, a visible border or outline appears around the element, signaling that it is the current focus point.

Best Practices

  • Ensure High Contrast: The focus indicator should be clear and have enough contrast with the surrounding content. Avoid subtle changes that can be hard to notice, especially for users with low vision or color blindness.
  • Custom Focus Styles: Use CSS to enhance the focus indicator (:focus, :focus-visible) without removing it completely. Removing focus indicators entirely (outline: none;) can make navigation impossible for keyboard users.

Focus Indicator and Non-Text Contrast Requirements

Focus indicators must also meet non-text contrast requirements to ensure they are easily distinguishable from surrounding elements. For more information, please see: Non-Text Contrast Guide.

Following these guidelines will ensure that your site is accessible to all users, particularly those relying on keyboard navigation.

Designing for Focus Visible Accessibility

  • Define Visible Focus Styles: Ensure that all interactive elements (links, buttons, form fields) have a clearly defined focus style to distinguish them when they receive focus via keyboard navigation.
  • Contrast for Focus Indicators: Design focus indicators with sufficient contrast against their background to make them easily noticeable. A contrast ratio of at least 3:1 against the surrounding elements is recommended.
  • Consistent Focus Indicators: Maintain consistency in the appearance of focus indicators across the interface to help users predict and understand how to navigate your design.
  • Do Not Rely on Color Alone: Design focus indicators that do not rely solely on color changes, considering users who may have difficulty perceiving color differences. Consider using outlines, shadows, or other non-color design elements.

Focus Visible

  • All interactive elements must have a visible focus indicator.
  • The focus indicator must have a 3:1 minimum contrast ratio with the adjacent element color AND the page background color.
  • Recommendation: Use a double outline with a dark and light color and outline offset to give some space between outline and element.

Implementing Focus Visible Accessibility

  • Implement Visual Focus Indicators: Ensure all interactive elements display a clear visual focus indicator that distinguishes them from non-focused elements.
  • Custom Focus Styles: Use CSS to create custom focus styles that are easily visible and distinguishable from the element's default state.
  • Consider Dual Colors: When implementing a global focus style consider using a dual color system (with auto) for the outline to ensure focused visible contrast requirements are met when the element is against light backgrounds and dark backgrounds.
  • Test for Keyboard Navigability: Regularly test your application using only a keyboard to ensure that all interactive elements are reachable and functional.
  • Consider High Contrast Modes: Ensure focus indicators are visible even in high contrast modes, enhancing accessibility for users with visual impairments.

By integrating clear and distinct focus indicators in the development process, you enhance accessibility for keyboard users, aligning with WCAG 2.4.7 Focus Visible criteria and improving overall usability for a wider audience.
