Focus Visible

Every user interface that can be operated using a keyboard should have a mode where the keyboard focus indicator is clearly visible.

  1. Request examples, if not provided of elements in their focus state.
  2. Visual Inspection: Examine your Figma design carefully. Ensure that interactive elements (buttons, links, form fields, etc.) have clear and distinct visual styles that indicate their interactive nature. This can include using color, borders, shadows, or other visual cues.

  • All interactive elements must have a visible focus indicator.
  • The focus indicator must have a 3:1 minimum contrast ratio with the adjacent element color AND the page background color.
  • Recommendation: Use a double outline with a dark and light color and outline offset to give some space between outline and element.

Note:

Please also test the color contrast of the indicator to ensure: the focus indicator contrast between the indicator and background must maintain a minimum color contrast ratio of 3:1 and color contrast between the indicator and the adjacent focused element must also maintain a minimum of 3:1 ratio. Please refer to Non-Text Color Contrast for more information.
